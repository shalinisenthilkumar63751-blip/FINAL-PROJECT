## 🚀 Running the Application

**Prerequisites:** Python, Streamlit

1. Open a terminal in the project folder:
   `cd python_project`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the Streamlit app:
   `python -m streamlit run app.py`
